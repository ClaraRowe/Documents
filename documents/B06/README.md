# 人生攻略第六部-如何才能活得更久？

## 前言

1. 我没有任何医学背景，以下所有suggestion(建议)仅从我经过审查后认可的作者所著的书籍中摘抄。
2. 我只保留结论，去除了原理解释和实验数据，你好奇可以去问ai，写原理就太冗长，不方便查看。

## 第零步：健康的前提是合理的体重

1. 询问AI你的BMI指数是多少，问他在同年龄段数据是否合理。
2. 然后该减肥减肥，符合条件就看下一步。

## 第一步：保持肌龄

1. 寻找一些自拍检测皮肤状态的软件，多测几个软件，把问题总结下来。
2. 去视频平台搜索解决方案，去问ai寻找解决方案，去问京东互联网医生寻求答案，去线下医院寻求专业解决方案。
