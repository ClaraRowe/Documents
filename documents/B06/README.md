## 第二步：记住这些建议

1. 应当少食用smoked(烟熏)和腌制肉类，因其可能增加某些肿瘤的发生风险。
2. 少盐少油，控糖限酒。我国居民普遍摄入过多食盐、烹调油和fat(脂肪)，这是导致肥胖、心脑血管等chronic(慢性)病发病率居高不下的重要原因。建议培养清淡饮食习惯，成年人每日食盐摄入不超过5g、烹调油25-30g，并避免过量摄入animal(动物性)油脂和饱和脂肪酸。 （来源：《中国居民膳食指南（2022）》）
3. 远离吸烟者。二手烟的危害范围可达9米半径，并能在空气中持续悬浮数小时。三手烟残留物会附着在衣物、家具等表面，持续释放有害化学物质，长期接触可能增加致癌风险。（来源：香港卫生署）
4. 吸烟会导致黑头粉刺发炎，进而形成acne(痤疮)丘疹。这些症状会影响头发毛囊、腋窝、生殖器、肛门部位的腺体，甚至蔓延至腹股沟，引发反常性痤疮（Akneinversa）。此类痤疮会出现严重的炎症性肿胀和pustules(脓疮)，并伴有疼痛。"Inversa"是指发生在skin(皮肤)褶皱处、会引起脓疮和疖子的痤疮。（来源：《皮肤的秘密》）
5. 长期drinking(饮酒)会使男性睾酮水平下降，使男子女性化，失去大量体毛（少于头发），男子汉气概也会缩水，甚至长出breast(乳房)。过多的alcohol还会损害控制面部血管的神经，导致血液循环系统无法对日常的冷暖、压力、伤害和恼怒做出正常反应。
6. 反式脂肪会增加heart(心)梗死和中风风险，导致bloodpressure(血压)升高，并可能引发癌症、2型糖尿病、皮肤早衰和过敏。此外，transfats(反式脂肪)还会促进皮脂分泌和角质化，导致毛孔堵塞和痤疮。由于transfat(反式脂肪)的这些严重危害，美国已在食品生产中禁止使用。（来源：《皮肤的秘密》）
7. 过量摄入addedsugar(添加糖)会增加龋齿和超重风险。建议避免或少喝含糖饮料，每日糖摄入量应控制在50g以下，最好不超过25g。儿童青少年、孕妇、乳母不宜drink(饮酒)；成年人如饮酒，每日alcohol(酒精)摄入量不应超过15g。
